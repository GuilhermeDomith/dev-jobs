Esta plataforma é apenas um protótipo, foi desenvolvido como trabalho final do curso de mineração de dados aplicada, do Instituto Federal do Sudeste de Minas Gerais - Campus Barbacena.

A plataforma tem o objetivo de facilitar para os usuário a forma de encontrar vagas de emprego e ter opções para aprimorar seus conhecimentos. Com este objetivo, utiliza-se técnicas de mineração de dados para encontrar vagas, competêcias e sugerir cursos de uma forma mais eficiente, utilizando similaridade de cossenos e correlação entre os conteúdos.
